# Network-Automation-using-a-Bot
This project simulates a real-world home/small office network setup where end devices communicate through a router and connect to the internet (represented by Cloud-PT). The lab covers fundamental networking concepts including IP addressing, interface configuration. To monitor this a Bot agent is used which collects all the data.

What's Inside:

Network topology — ASCII diagram showing connections
Devices & connections table — exact cables and ports
IP addressing table — all IPs in one place
Step 1 — Router CLI commands with explanations
Step 2 — PC1 IP configuration
Step 3 — Cloud-PT DSL setup
Troubleshooting section — reset PC, reset router, ROMMON fix
CLI mode reference — all modes and how to exit
Key commands table — full cheat sheet
Concepts covered — great for recruiters/portfolio
